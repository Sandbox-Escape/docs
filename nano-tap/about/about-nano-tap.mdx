---
title: "What is NanoTap?"
---

NanoTap is a mini-app service that allows you to bypass signing restrictions on iOS and Android (Android Version Coming Soon) that allows users to install your app via a 10 digit code called a NanoCode, NFC Tag or QR Code. \
\
Downloads for NanoTap are currently unsigned, meaning you have to install NanoTap manually with your own certificates. Please go to the next page to see how!
